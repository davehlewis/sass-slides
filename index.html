SASS & SCSS

  • Been around for 12 years
  • CSS Preprocessor
  • Requires Build process
  • 2 Flavours - SASS and SCSS

SASS

$color: red
$coloralt: green

a
  color: $color
  &:hover
    color: $coloralt

SCSS

$color: red;
  $coloralt: green;
  
  a {
    color: $color;

    &:hover {
      color: $coloralt;
    }
  }

Why use SASS?

  • Variables
  • Nesting
  • Partials & Import
  • Mixins
  • Includes & Extends
  • Operators
  • Feature not changed too much
  • Variables to save on repetition, colors, sizes etc.
  • Partials and Import allow for modular code. Button Module, Layout module etc.
  • A mixin lets you make groups of CSS declarations that you want to reuse throughout your site.
  • ets you share a set of CSS properties from one selector to another
  • Operators - maths. Widths, heights, paddings etc.

It's still popular

  • Bootstrap
  • Bulma
  • Foundation
  • UIKit
  • Many offer LESS versions also
  • Allows overrides of variables for customisation
